# A股智能选股APP PRD

## 1. 产品定位
A股智能选股APP面向个人投资者、炒股高手和基金/ETF投资者，提供多视角选股、ETF筛选、组合配置、回测验证和风险控制能力，帮助用户把技术面、基本面、市场情绪与基金经理视角整合为可解释的投资决策参考。

> 免责声明：本产品仅提供数据分析和投资研究工具，不构成任何投资建议或收益承诺。

## 2. 目标用户
- **普通投资者**：希望快速发现符合条件的A股、ETF和主题机会。
- **进阶交易者**：需要自定义技术指标、基本面条件、情绪指标和权重评分。
- **基金/ETF配置用户**：关注行业轮动、资产配置、费率、折溢价、流动性和回撤控制。
- **投研/运营团队**：需要沉淀可复用策略、回测结果和榜单内容。

## 3. 核心目标
1. 建立跨平台移动端体验，优先交付MVP。
2. 支持多条件选股：技术指标、基本面、市场情绪和自定义权重。
3. 支持ETF筛选和组合配置：跟踪误差、规模、费率、折溢价率、行业轮动。
4. 提供风控体系：最大回撤、夏普比率、波动率、VaR等指标。
5. 提供历史回测、参数优化和样本外验证。
6. 满足合规要求：数据版权、投顾牌照边界、风险提示和免责说明。

## 4. 功能范围
### 4.1 MVP阶段
- 股票池管理：A股基础股票池、行业/板块标签、可交易状态。
- 策略条件：均线、MACD、KDJ、RSI、布林带、量价指标。
- 基本面筛选：PE、PB、ROE、营收增速、负债率。
- 市场情绪：北向资金、龙虎榜、板块轮动、涨停强度。
- 多条件组合：AND/OR逻辑、权重评分、排序和结果解释。
- 移动端核心页面：策略配置、选股结果、个股详情、风险提示。

### 4.2 V1.0
- ETF模块：ETF筛选、ETF详情、组合再平衡。
- 桌面端/管理端：策略管理、回测配置、运营榜单。
- 回测系统：历史回测、参数优化、样本外验证。

### 4.3 V2.0
- AI选股助手：自然语言配置策略、策略解释、结果摘要。
- 社区：策略分享、订阅、评论和付费内容。
- 付费体系：会员权益、策略市场、数据增值服务。

## 5. 用户流程
1. 用户选择股票/ETF市场与时间范围。
2. 用户配置技术面、基本面、情绪面或组合模板。
3. 系统拉取行情和因子数据，执行策略评分。
4. 用户查看候选清单、命中原因、风险指标和历史表现。
5. 用户保存策略、订阅提醒或进入回测。

## 6. 非功能需求
- **性能**：常规选股查询P95小于3秒；行情推送延迟目标小于2秒。
- **可用性**：核心服务可用性目标99.5%。
- **安全**：鉴权、限流、审计、敏感配置隔离。
- **合规**：全链路风险提示，不展示确定性收益承诺。
- **可解释性**：每只候选股票必须展示命中条件和评分来源。

## 7. 验收标准
- 用户可以完成至少10个内置策略模板的选股。
- 系统可以输出股票评分、命中规则和风险摘要。
- 测试数据下策略引擎结果稳定、可复现。
- 文档覆盖产品需求、技术架构、实现方案和测试计划。
